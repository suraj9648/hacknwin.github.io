How to Install Apache24



Step 1:    Download  and install Apache24 for free.




Step 2:  Extract Downloaded file in C Drive


After Extract in C Drive



Step 3:  Open httpd.conf File from conf folder



Step 4:  Now Edit httpd.conf File with your favourite text editor
Now make this changes on httpd.conf file Define SRVROOT "c:/Apache24" and Listen 8080


Now go to the last line and add this Code
 AddHandler application/x-httpd-php .php
 AddType application/x-httpd-php .php .html
 LoadModule php7_module "c:/php7/php7apache2_4"
 PHPIniDir "c:/php7"



After make Changes save the file

Step 5:  Open CMD As Administrator and go to your apache24 bin folder. For that type cd C:\Apache24\bin in CMD and press enter. Now you are in bin folder



Step 6: Installing Apache24
For that type httpd.exe -k install in CMD



Step 7: Run Apache24
For that go to your bin folder than double click on ApacheMonitor.exe



Step 8: Now Apache24 is Running




Step 9: Now You are ready to host Files
Just host your files in htdocs Folder



How to use


Open Your Browser and type in URL 127.0.0.1:8080 OR localhost:8080
