How To Install PHP7



Step 1:    Download  and install PHP7.





Step 2: Create a php7 folder in C Drive




Step 3: Now Extract PHP7 Zip file in php7 Folder




Step 4: Now copy php.ini-development and paste it in same folder. After Copy your get a new file php - Copy.ini-development Now Rename this file to php.ini .




Step 5: Now Open php.ini with any text editor. And find this Extension. Than remove ; before extension_dir




After make changes save the file


Step 5: Now Set Environment_variables For both User and Adminstrator account.

Search Variables in Search bar


Click Environment Variables


Now create new value in Path. For both user and adiminstrator account


Now create a New path and type C:\php7 and click Ok.


Now you can check your php7 is running correctly or not
For that type php -v in cmd



Remember: Add this path for both user and Adminstrator account
